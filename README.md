Приложение "Список задач" должно сохранять задачи, которые должны быть решены в конкретные сроки и их краткое описание. Должна быть возможность включать разовые задачи и регулярные (ежедневно, еженедельно и т.д.). При выполнении задачи ставится отметка, возможно комментарий, если выполнено не в срок. Если срок меняется, обоснование - причины.

Сохранять информацию можно в файле.
